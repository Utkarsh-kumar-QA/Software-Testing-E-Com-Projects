# OpenCart E-Commerce Manual Testing Project

## Project Overview

This project demonstrates Manual Testing activities performed on the OpenCart E-Commerce application. The objective is to validate the functionality, usability, reliability, and overall quality of the application through various testing techniques.

## Application Details

* Application Name: OpenCart
* Domain: E-Commerce
* Testing Type: Manual Testing
* Environment: Web Application
* Browser Support: Chrome, Firefox, Edge
* URL: https://demo.opencart.com/

---

## Project Objectives

* Verify all major functionalities of the OpenCart application.
* Identify and report defects.
* Ensure the application meets business requirements.
* Validate end-to-end user workflows.

## Sample Test Case

**Test Case ID:** TC_LOGIN_001

**Test Scenario:** Verify user login with valid credentials.

**Steps:**

1. Navigate to Login Page.
2. Enter valid email address.
3. Enter valid password.
4. Click Login button.

**Expected Result:**
User should be successfully logged in and redirected to My Account page.

**Actual Result:**
User successfully logged in.

**Status:**
Pass
